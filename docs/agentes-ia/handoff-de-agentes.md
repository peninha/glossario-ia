# Handoff de Agentes

**Agent Handoff**

Transferência de controle entre [agentes](../agentes-ia/agente.md) especializados em [sistemas multiagentes](../agentes-ia/sistemas-multiagentes.md). Quando um agente determina que outra especialidade é necessária, ele passa a tarefa (e o contexto relevante) para o agente adequado. Mecanismos comuns incluem: handoff via [ferramenta](../agentes-ia/ferramentas.md), condições avaliadas por [LLM](../ia-generativa/modelos-de-linguagem-grande-porte.md), [roteamento](../agentes-ia/roteamento-de-agentes.md) baseado em contexto e comportamento pós-tarefa (fallback).
