# Agente de Navegador

**Browser Agent**

[Agente](../agentes-ia/agente.md) de IA que automatiza a navegação web, compreendendo conteúdo de páginas, adaptando-se a mudanças de layout e executando workflows multi-etapa — substituindo automação frágil baseada em scripts (como Selenium). Utiliza capacidades de [uso de computador](../agentes-ia/uso-de-computador.md) para interagir com sites que não possuem [API](../infraestrutura-processos/api.md). Exemplos atuais: OpenAI Operator, Google Project Mariner, Anthropic Claude for Chrome e Perplexity Comet.
