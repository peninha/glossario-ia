# Uso de Computador

**Computer Use**

Capacidade de [agentes](../agentes-ia/agente.md) de IA interagirem com interfaces gráficas de computador como um humano faria: movendo o mouse, clicando em elementos, digitando texto e lendo a tela via capturas de imagem. Inaugurado pela Anthropic em outubro/2024, permite que agentes atuem em aplicações que não expõem [APIs](../infraestrutura-processos/api.md), ampliando drasticamente o [ambiente](../agentes-ia/ambiente.md) acessível. É a base tecnológica dos [agentes de navegador](../agentes-ia/agente-de-navegador.md).
