# Agente de Código

**Coding Agent** · *SWE Agent*

[Agente](../agentes-ia/agente.md) especializado em engenharia de software que autonomamente lê documentação, escreve código, executa testes, faz debug e propõe correções. Opera em [loop de agente](../agentes-ia/loop-de-agente.md) com [ferramentas](../agentes-ia/ferramentas.md) como leitura/escrita de arquivos, terminal, busca semântica e navegador. Exemplos: Claude Code, GitHub Copilot Agent, Cursor Agent, Devin e OpenHands. Relaciona-se com a prática de [vibe coding](../habilidades-praticas/vibe-coding.md), em que o usuário direciona o agente criativamente sem necessidade de programar diretamente.
