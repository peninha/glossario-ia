# Protocolo A2A

**Agent-to-Agent Protocol**

Protocolo aberto criado pelo Google (abril/2025) para comunicação entre [agentes](../agentes-ia/agente.md) de diferentes frameworks e fornecedores. Usa JSON-RPC 2.0 sobre HTTPS e define conceitos-chave como *Agent Card* (metadados JSON de identidade e capacidades), *Task* (interação com ciclo de vida), *Message* (turnos de comunicação) e *Artifact* (saídas geradas). Complementa o [MCP](../infraestrutura-processos/mcp.md): enquanto o MCP conecta agentes a [ferramentas](../agentes-ia/ferramentas.md), o A2A conecta agentes entre si como pares. Conta com mais de 150 organizações contribuintes.
