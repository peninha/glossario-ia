# Saída Estruturada

**Structured Output**

Capacidade de [modelos de linguagem](../ia-generativa/modelos-de-linguagem.md) retornarem dados em formatos estruturados (JSON, XML ou schemas definidos) em vez de texto livre. Essencial para integração com sistemas, [chamada de ferramentas](../agentes-ia/chamada-de-ferramentas.md) e pipelines agênticos, pois garante que a saída do modelo possa ser processada programaticamente. Todas as grandes APIs (OpenAI, Anthropic, Google) oferecem modos de saída estruturada com validação de schema.
