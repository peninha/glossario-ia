# Fundamentação

**Grounding**

Técnica de conectar as saídas de um [modelo de linguagem](../ia-generativa/modelos-de-linguagem.md) a fontes verificáveis de informação — documentos, bases de dados, resultados de busca — para reduzir [alucinações](../ia-generativa/alucinacao.md). Inclui fundamentação via busca web (o modelo cita páginas), via documentos internos (similar a [RAG](../ia-generativa/rag.md)) e via [ferramentas](../agentes-ia/ferramentas.md) (o modelo consulta APIs para dados em tempo real). Diferencia-se de RAG por ser um conceito mais amplo que abrange qualquer mecanismo de ancoragem factual, não apenas recuperação de documentos.
